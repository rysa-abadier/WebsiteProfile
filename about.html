Quarter 4 - Assignment #1

Basic CSS Styling

Your task is to create a basic webpage using HTML and apply CSS styling to it. You can create a page on any topic of your choice, such as your favorite hobby, a travel destination, or a personal blog.

Refer to template

Requirements:

  • The webpage should include at least two different sections, such as a header and a main content area.
  • Apply a background color or image to the webpage.
  • Use differnt font styles, sizes, and colors to make the content more visually appealing.
  • Apply CSS rules to adjust the spacing between different elements on the page.
  • Include a navigation menu and apply styling to it.
